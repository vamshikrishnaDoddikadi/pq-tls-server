Built for Production

🔐

ML-KEM-768 (FIPS 203)

NIST-standard post-quantum key exchange in hybrid mode with X25519. Clients without PQ support fall back to classical X25519.

⚡

Single Binary

Zero runtime dependencies beyond OpenSSL + liboqs. Everything embedded — management UI, certificates, configuration.

🔄

Crypto-Agility

Pluggable provider registry. Swap algorithms, add new KEMs, and update policies at runtime — no recompilation needed.

📊

Management Dashboard

Real-time HUD with connection throughput, PQ adoption ratio, live log viewer, certificate management, and more.

🛡️

Security Built In

Per-IP rate limiting, CIDR-based ACLs, hot certificate reload, structured JSON logging. Runs as non-root user.

📈

Prometheus Metrics

Full metrics endpoint for Grafana dashboards. Track connection counts, handshake failures, PQ vs classical ratios.

Get Started in 60 Seconds

# Run PQ-TLS Server with Docker
docker run -d \
  -p 8443:8443 -p 9090:9090 \
  -v ./certs:/etc/pq-tls-server/certs:ro \
  vamshikrishna/pq-tls-server \
  --backend your-app:8080 --health-port 9090

# Test with curl
curl --cacert certs/ca.crt https://localhost:8443/

# Open dashboard at http://localhost:9090
