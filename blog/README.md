## Tips for Posting

1. Post one per week — don't spam
2. Engage in comments immediately after posting
3. Link to the GitHub repo at the bottom
4. For HN: post early morning US time (6-8am PT) for best visibility
5. For Reddit: always mention you're the author to avoid self-promotion penalties
